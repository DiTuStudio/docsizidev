# Vimall (ReactJs)

## Environment confirmation

```bash
$ yarn -v
1.22.17

$ node -v
v14.17.5

$ npm -v
6.14.14
```

## Run in development

```bash
cd <react-native-project-root>
yarn install
```

If you are using npm without yarn

```bash
cd <react-native-project-root>
npm install
```

Then, start with command below

```bash
yarn start
```

or

```bash
npx start
```
